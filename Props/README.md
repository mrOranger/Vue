# PROPS

In Vue.js viene indicato con il termine **props**, abbreviazione di _properties_, una serie di elementi che vengono _"passati"_ da una componente ad un'altra.

Questo meccanismo risulta molto efficacie qualora si vogliano propagare delle informazioni a modo di cascata tra componenti.

Usando questo meccanismo, si tiene fede al concetto di separazione delle componenti all'intero di una pagina costruita con Vue. Inoltre, in questa maniera non sar√† necessario duplicare il codice per riferirsi ad elementi diversi che condividono una stesso schema con informazioni diverse.
